Snake Game (HTML, CSS, JavaScript)

A modern, responsive Snake Game built using Vanilla JavaScript and HTML5 Canvas, featuring smooth gameplay, pause/resume functionality, and a polished UI with overlay screens.

🚀 Features
🎮 Classic Snake gameplay using Canvas rendering
⏯️ Pause & Resume functionality (continues from same position)
🔄 Restart game anytime
🟢 Start screen before gameplay begins
💀 Game Over overlay with final score display
🏆 High Score tracking using localStorage
🎉 New High Score detection
🌫️ Blur UI overlay for pause and game over states
⚡ Smooth game loop using requestAnimationFrame
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Canvas API

🎯 Controls
⬆️ Arrow Up → Move Up
⬇️ Arrow Down → Move Down
⬅️ Arrow Left → Move Left
➡️ Arrow Right → Move Right
⏸️ Pause Button → Pause Game
▶ Resume Button → Continue Game
